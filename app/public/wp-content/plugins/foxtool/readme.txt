== Description ==

Foxtool is a product developed based on the real needs of managing WordPress websites. After a period of development, Foxtool has become an indispensable plugin for website administrators.

The plugin boasts notable features including:

- Website optimization
- Advanced security
- Management tools
- Customizable display
- Powerful media management
- Handy tools for Posts and Pages
- Mail application
- Woocommerce convenience
- User permissions
- Customizable WordPress login
- Integration with Google services
- Chat functionality
- Information management

Additionally, there are many other useful features that help save time for users. All are optimized to ensure stable and fast performance for your WordPress website

== Installation ==

=== From within WordPress ===

1. Visit 'Plugins > Add New'
2. Search for 'Foxtool'
3. Activate 'Foxtool' from your Plugins page.

=== Manually ===

1. Upload the 'foxtool' folder to the '/wp-content/plugins/' directory
2. Activate the 'Foxtool' plugin through the 'Plugins' menu in WordPress
3. Go to "after activation" below.
